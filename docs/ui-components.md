# Modified UI Components

The following UI components have been customized, specifically by adding the `cursor-pointer` class to interactive triggers to improve user experience.

## List of Components

- **Accordion** (`AccordionTrigger`)
- **AlertDialog** (`AlertDialogTrigger`)
- **Checkbox**
- **Dialog** (`DialogTrigger`)
- **Drawer** (`DrawerTrigger`)
- **DropdownMenu** (`DropdownMenuTrigger`, `DropdownMenuItem`, etc.)
- **RadioGroup** (`RadioGroupItem`)
- **Select** (`SelectTrigger`)
- **Sheet** (`SheetTrigger`)
- **Switch**
- **Tabs** (`TabsTrigger`)
